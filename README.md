*** First C++ Project — Student Instructions ***

This project introduces:
  C++ class design (Student)
  Project structure (.h, .cpp, main.cpp)
  std::vector
  Sorting
  CMake
  CLion workflow

To clone, build, and execute this project:
1. Open CLion
2. On the welcome screen, choose Get from Version Control
3. Paste this URL: https://github.com/juliancochran/FirstCPPProject01.17.26.git
4. Choose a location on your computer
5. Click Clone
6. CLion will automatically open the project.

Next, understand the project files. We'll discuss Student.h, Student.cpp, main.cpp, and CMakeLists.txt in the project.

To build and run the program:
1. In CLion, click Build > Build Project or click the hammer icon
2. Click the Run ▶️ button (top-right)
